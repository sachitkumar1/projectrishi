Drop your photos in this folder, then reference them in lib/content.ts.

Example:
  1. Save a file here:        public/images/hero-village.jpg
  2. In lib/content.ts set:   heroImage: "/images/hero-village.jpg"

Subfolders:
  projects/   project-specific cover + highlight photos
  gallery/    the "Life at Project RISHI" gallery on the About page
